#概要
このプログラムは、基礎的なアルゴリズムを学びたい、
自身のスキルを試したいという気持ちで作成しました。
RubyとGosuライブラリを使用して、ゲームのロジックや描画処理、当たり判定などを実装しています。

#環境
Ruby : 3.3.7(2025/01/15)
使用したライブラリ : Gosu

#使い方
1.Ruby環境を構築します。
2.`main.rb`を実行します。
3.実行すると、ウィンドウが開き、ブロック崩しゲームが起動します。
